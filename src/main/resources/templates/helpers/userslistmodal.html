Blocked users list

This list contains informations about subnetwork users, which will be used in blocking specified user.

Columns explanations:

User ip - text field containing valid ip address of specified user.

Active - boolean value indicating if user could be possibly blocked.

Time allowed - boolean value indicating if list is active only in specified time. True means that user is not blocked in time period specified by Start time and End time.

Start time - beginning of time period when user is not being blocked while Time allowed is true.

End time - nd of time period when user is not being blocked while Time allowed is true.

Browsing time limited - boolean value indicating if values specified in Used time and Duration could lead to block user.

Duration - integer value specified how many minutes per time not blocked period may be used by the user before browsing time limited block.

Used time - integer value specified how many minutes per time not blocked period was wasted by user.
