Words list

This list contains collections of words, which will be used in filtering process. Each collection could be limited during time.

Columns explanations:

List name - text field containing name of list specified by user. Name should group all contained words in same context like 'Bad words' or 'Obscenities'.

Active - boolean value indicating if list i currently being used by filtering system. If list is active, contained words could lead to block website.

Timed - boolean value indicating if list is active only in specified time. True means that words from list are used in time period specified by Start time and End time.

Start time - beginning of time period when words list is being used in filtering process if Timed is true.

End time - end of time period when words list is being used in filtering process if Timed is true.
