Transsmision types

This list contains collections of mime content-type divided into four sets: -Video_streams - blocks mime types using in video streaming,
-Video - block mime types using in sending video data,
-Audio - block mime types using in sending audio data,
-Image - block mime types using in sending images.

Columns explanations:

Transmission type - text field containing name od specified set of mime types.

Active - boolean value indicating if transmission type is currently being used by blocking system. True means that set of mime types are prohibited.

Timed - boolean value indicating if set of mime types is active only in specified time. True means that words from list are used in time period specified by Start time and End time.

Start time - beginning of time period when set of mime types is being used in filtering process if Timed is true.

End time - end of time period when set of mime types is being used in filtering process if Timed is true.
