Domains list

This list contains collections of domains blocked for every user. Each collection could be limited during time and could have one of two modes.

Columns explanations:

List name - text field containing name of list specified by user. Name should describe list's contents like 'Social networking websites', 'Games' etc.

Active - boolean value indicating if list is currently being used by blocking system. If list is active, it's domains are prohibited.

Timed - boolean value indicating if list is active only in specified time. If true, domains are blocked in time period specified by Start time and End time.

Start time - beginning of time period when domains are blocked if Timed is true.

End time - end of time period when domains are blocked if Timed is true.

Mode - it describes the way blocking system treats domains list. Blacklist means that specified domains are blocked, Whitelist means that domains are permitted.
